Для составления конфигурации Mihomo можно воспользоваться:
1. Онлайн генератором web4core - https://spatiumstas.github.io/web4core
2. Составить вручную используя мануал - https://ssclash.notion.site/Super-Simple-Clash-15989188f6b48051a97fc887adea736a (может не открыться из-за ограничений хостингов со стороны Роскомнадзора)
3. Использовать готовые конфигурации где нужно заменить строки в прокси на свои:
   - [Cloudflare WARP](https://github.com/Internet-Helper/mixomo-openwrt/blob/main/files/Mihomo/Cloudflare%20WARP/config.yaml)
   - [Cloudflare WARP + Proton](https://github.com/Internet-Helper/mixomo-openwrt/blob/main/files/Mihomo/Cloudflare%20WARP%20%2B%20Proton/config.yaml)
   - [Cloudflare WARP + Proton + VLESS](https://github.com/Internet-Helper/mixomo-openwrt/tree/main/files/Mihomo/Cloudflare%20WARP%20%2B%20Proton%20%2B%20VLESS)
   - [Cloudflare WARP + Proton + Подписка](https://github.com/Internet-Helper/mixomo-openwrt/tree/main/files/Mihomo/Cloudflare%20WARP%20%2B%20Proton%20%2B%20%D0%9F%D0%BE%D0%B4%D0%BF%D0%B8%D1%81%D0%BA%D0%B0)
   - [Cloudflare WARP + VLESS](https://github.com/Internet-Helper/mixomo-openwrt/tree/main/files/Mihomo/Cloudflare%20WARP%20%2B%20VLESS)
   - [Cloudflare WARP + Подписка](https://github.com/Internet-Helper/mixomo-openwrt/tree/main/files/Mihomo/Cloudflare%20WARP%20%2B%20%D0%9F%D0%BE%D0%B4%D0%BF%D0%B8%D1%81%D0%BA%D0%B0)
   - [VLESS](https://github.com/Internet-Helper/mixomo-openwrt/tree/main/files/Mihomo/VLESS)
   - [Подписка](https://github.com/Internet-Helper/mixomo-openwrt/tree/main/files/Mihomo/%D0%9F%D0%BE%D0%B4%D0%BF%D0%B8%D1%81%D0%BA%D0%B0)

Бесплатную конфигурацию Cloudflare WARP можно получить [здесь](https://warp-generator.github.io/warp/) илли [здесь](https://generator-warp-config.netlify.app/) (может не открыться из-за ограничений хостингов со стороны Роскомнадзора).<br>
Бесплатную конфигурацию Proton можно получить здесь - [инструкция](https://help-guide.notion.site/Proton-1f72684dab0d80b481bdfb1df9148e0c) (может не открыться из-за ограничений хостингов со стороны Роскомнадзора).
